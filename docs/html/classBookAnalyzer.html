Подробное описание

Класс для анализа прочитанных книг читателями.

Использует класс UnorderedSet для хранения книг и выполнения операций над множествами. Определяет три категории книг: прочитанные всеми, прочитанные некоторыми, не прочитанные никем.

Методы

◆ Analyze()

void BookAnalyzer::Analyze ( )
inline

Выполняет анализ прочитанных книг.

Определяет три категории книг:

  1. Книги, прочитанные всеми читателями
  2. Книги, прочитанные некоторыми читателями (но не всеми)
  3. Книги, которые никто не прочитал
